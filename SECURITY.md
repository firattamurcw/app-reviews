# Security Policy

## Supported Versions

The following versions of `app-reviews` are currently being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1.0 | :x:                |

## Reporting a Vulnerability

We take the security of `app-reviews` seriously. If you believe you have found a security vulnerability, please report it through [GitHub Security Advisories](https://github.com/firattamur/app-reviews/security/advisories/new). This ensures your report remains private until a fix is available.

Please include:

1. A detailed description of the vulnerability.
2. Steps to reproduce the issue (a proof-of-concept script is highly appreciated).
3. The version(s) of `app-reviews` and the environment (OS, Python version) where the issue was found.

We will acknowledge your report within 48 hours and provide a timeline for a fix if the vulnerability is confirmed. We request that you follow responsible disclosure practices and give us time to address the issue before making any information public.
